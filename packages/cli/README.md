# @staffforge/cli

Universal installer for [StaffForge AI Agent Framework](https://github.com/mjuliac/StaffForge-AI-Agent-Framework).

```bash
npx @staffforge/cli
```

Interactive prompts for platform, agent, and location. Or configure with flags:

```bash
npx @staffforge/cli --platform opencode --agent orchestrator
```

Supports: OpenCode, Claude Code, Cursor, Copilot, Aider, Gemini CLI, and all.
